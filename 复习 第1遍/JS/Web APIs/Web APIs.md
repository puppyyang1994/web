7. 环境变量：JS允许在函数体内部，引用当前环境的其他变量
   需要一种机制，能够在函数体内部获得当前的context运行环境。所以`this`就出现了，它设计目的就是在函数体内部，指代函数当前的运行环境
   ```
   let f = function(){
      console.log(this.x) //this.x 就是指当前运行环境的x
   }
   ```
