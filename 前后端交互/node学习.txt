浏览器中的JS运行环境（代码运行所必需的必要环境）
每一个浏览器都是运行环境（以Chrome运行环境为例）
V8解析引擎 +内置API （dom bom canvas XMLhttprequest JS内置对象）
V8解析引擎负责解析和执行JS代码
内置API是运行环境（也就是浏览器）提供的特殊接口，只能在所属的环境中被调用


JS能否做后端开发（Java Python PHP）

Node.js是什么： 是一个基于Chrome v8引擎的 JavaScript运行环境


Node.js中的JavaScript运行环境
Node.js运行环境：
V8引擎 + 内置API（fs path http JS内置对象  querystring）

注意：浏览器是JavaScript的前端运行环境
Node.js是JavaScript的后端运行环境， Node.js中无法调用DOM和BOM等浏览器内置API

基于 restify 可以快速构建API接口项目
读写和操作数据库 创建使用的命令行工具辅助开发
